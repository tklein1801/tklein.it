# tklein.it

## Getting started

1. Clone the repo

   ```bash
   git clone git@github.com:tklein1801/tklein.it.git
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Set environment variables (as defined in the `./.env.example`)

   ```bash
   # Start the development server
   npm run dev
   ```
